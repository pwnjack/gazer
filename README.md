# gazer.io
Bug tracking made easy

- Projects
	- Branches
- Users
	- Groups
	- Role
- Bugs/Issues
	- Title
	- Comments
	- Priority (Low, Normal, High)
	- Assignee (User, Group)
	- File attachment (Screenshot, Document)
	- Status (Open, Working, Solved)
	- Labels (Wont Fix, ?)
- Github integration
- Time tracking (from status to status)
- Invoicing
